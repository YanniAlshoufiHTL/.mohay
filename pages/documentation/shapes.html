Shapes

Rectangles

To create rectangles, you use the "rect" keyword.

After 'rect' you need to add the starting point and the width & height of the rectangle.

Example

.r = rect (0, 0) 10 20

After rect you need to add a point and a radius.

Example

.c = circle (100, 100) 20

Arc

In .mohay you can draw parts of a circle.

This can be done by using the "arc" keyword.

It is followed by a starting point, radius, starting angle and end angle.

Example

.a = arc (3, 3) 10 20° 30°

Triangle

Triangles are created by using the "triangle" keyword.

After triangle you need to add three points.

Those will be the corners of your triangle

Example

.t = triangle (3, 3) (9, 5) (100, 53)

Polygon

Polygons are created by using the "polygon" keyword.

There are 2 ways of creating polygons.

  • Either you enter a middlepoint, number of corners, radius between middlepoint and corners and the rotation.
  • Or you just enter as many points as you like.

Example

.polygon1 = polygon (1, 1) 6 10 20°

.polygon2 = polygon (1, 1) (4, 10) (8, 16) (10, 100) (49, 28) (47, 19)
