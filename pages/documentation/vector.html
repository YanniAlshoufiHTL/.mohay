    Declaration

    Vectors are declared by using the "vector" keyword.

    Afterwards you need to enter two points.

    The difference between a line and a vector is that the vector is displayed with an arrow at the last point.

    In the example below the arrowhead would appear at the point (100, 2)

    Example

    var v = vector (3, 4) (100, 2)
